Data Types

1)NUMBER

Stores numeric values.

syntax:
NUMBER(precision, scale)
Precision → Total number of digits---- range upto 38 
Scale → Number of digits after decimal point----range upto 37


SAL NUMBER(10,2)

Example:

25000.50
1000.75
99999999.99

2)VARCHAR

Stores text values(char enclosed within single quotes'').
syntax:
varachar(size)


Example:

ALLEN
SMITH
BLAKE


3)DATE

Stores date only.

syntax:
date

formate:
'yyyy-mm-dd'


Example:

2025-06-01

4)TIMESTAMP

Stores date and time.

syntax:
timestamp;

format:

'yyyy-mm-dd hh:mm:ss.mss

Example:

2025-06-01 01:21:22.123

5)BOOLEAN

Stores logical values.

syntax:
boolean

Example:

TRUE
FALSE
