Warehouse provides compute power to run queries.

step-1:

Syntax
CREATE WAREHOUSE warehouse_name;

Example
CREATE WAREHOUSE MYWH;
CREATE WAREHOUSE MYWH2;


Step 2: Create Database

Database is a container that stores schemas.

Syntax
CREATE DATABASE database_name;
Example
CREATE DATABASE MYDB;
CREATE DATABASE MYDB2;


Step 3: Create Schema

Schema is a container inside a database.

Syntax
CREATE SCHEMA schema_name;
Example
CREATE SCHEMA MYSCHEMA;
CREATE SCHEMA MYSCHEMA2;

Data Types

1)NUMBER

Stores numeric values.

syntax:
NUMBER(precision, scale)
Precision → Total number of digits
Scale → Number of digits after decimal point


EID NUMBER
SAL NUMBER(10,2)

Example:

25000.50
1000.75
99999999.99

2)VARCHAR

Stores text values(char enclosed within single quotes'').
syntax:
varachar(size)

ENAME VARCHAR

Example:

ALLEN
SMITH
BLAKE


3)DATE

Stores date only.

syntax:
date

formate:
'yyyy-mm-dd'

DOJ DATE

Example:

2025-06-01
TIMESTAMP

Stores date and time.

JOIN_TIME TIMESTAMP

Example:

2025-06-01 01:21:22.123
BOOLEAN

Stores logical values.

IS_ACTIVE BOOLEAN

Example:

TRUE
FALSE
Step 6: Create Table
Syntax
CREATE TABLE table_name
(
 column_name datatype
);
Example
CREATE OR REPLACE TABLE EMP
(
 EID NUMBER,
 ENAME VARCHAR,
 SAL NUMBER(10,2),
 DOJ DATE,
 JOIN_TIME TIMESTAMP,
 IS_ACTIVE BOOLEAN
);
Real-Time Example
EMP Table

EID       → Employee ID
ENAME     → Employee Name
SAL       → Salary
DOJ       → Date of Joining
JOIN_TIME → Joining Timestamp
IS_ACTIVE → Employee Active Status

CURRENT_DATE()

Returns today's date.

SELECT CURRENT_DATE();

Example:

2026-06-09
CURRENT_TIMESTAMP()

Returns current date and time.

SELECT CURRENT_TIMESTAMP();

Example:

2026-06-09 16:30:25.123
Check Current Timezone

Purpose:

Set Indian Standard Time (IST)
